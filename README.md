# subtitlemod
`subtitlemod` is a quick and simple tool that modifies subtitle file timings.

## Supported Files
* .srt (SubRip Subtitle File)

## Current Feature Set
* CLI arguments supported (currently not implemented)
* Shift timings by constant value
